# 사용자 매뉴얼

## 전체 워크플로우

```
로그인 → 프로젝트 생성 → RFP 업로드 → AI 분석 → 방향성 선택 → 전략 수립
→ 목차 생성 → 섹션 작성 → 검토 → 가격 제안 → 산출물 출력
```

## 1. 로그인

1. 브라우저에서 시스템 접속
2. 아이디/비밀번호 입력 후 로그인
3. 최초 설치 시 `/setup`에서 Super Admin 생성

## 2. 프로젝트 생성

1. 대시보드에서 **새 프로젝트** 클릭
2. 프로젝트명 입력 후 생성

## 3. RFP 업로드

1. 프로젝트 → **RFP 업로드** 단계
2. RFP 파일(PDF, DOCX, HWP) 업로드
3. 업로드 완료 후 다음 단계로 이동

## 4. AI 분석

1. **분석 시작** 버튼 클릭
2. AI가 RFP에서 다음 항목을 자동 추출:
   - 사업 개요 (사업명, 발주기관, 예산, 기간, 목적)
   - 평가항목 및 배점
   - 요구사항 (기능/비기능/성능)
   - 추적성 매핑
   - 자격요건
   - 핵심 키워드
3. 실시간 진행 상황 표시
4. 추출된 내용 검토 및 수정 가능

## 5. 방향성 선택

1. AI가 3개의 제안 방향성을 생성
2. 각 방향성의 핵심 메시지, 차별화 포인트 확인
3. 가장 적합한 방향성 선택

## 6. 전략 수립

1. 선택한 방향성 기반으로 AI가 전략 수립
2. Win Theme, 차별화 요소, 경쟁력 분석 제공

## 7. 목차 생성

1. AI가 평가항목 기반 최적 목차 생성
2. 각 챕터의 제목, 페이지 배분 확인
3. 드래그앤드롭으로 순서 변경 가능
4. 평가항목-목차 매핑 관계 확인

## 8. 섹션 작성

1. **전체 생성** 또는 개별 섹션 생성
2. AI가 각 섹션의 본문 작성
3. 에디터에서 직접 수정 가능
4. 개별 섹션 **재생성** 가능

## 9. 검토

1. AI가 제안서 품질 검토 수행
2. 평가항목별 충족도, 요구사항 반영도 분석
3. 개선 제안 확인

## 10. 가격 제안

1. AI가 사업 규모에 맞는 가격 구조 생성
2. 인건비, 장비비, 경비, 간접비 항목 확인
3. 가격 경쟁력 분석

## 11. 산출물 출력

1. **Word 다운로드**: 전체 제안서 본문 (.docx)
2. **PPT 다운로드**: 발표용 요약 장표 (.pptx)
3. 커스텀 템플릿 선택 가능 (사전 등록 필요)

## 관리자 기능

### 사용자 관리
- 사용자 추가/수정/삭제
- 역할 부여: Super Admin, Admin, Manager, User

### AI 설정
- AI 프로바이더 선택 (Claude / GPT)
- 모델 선택
- API 키 설정 및 연결 테스트

### 브랜딩
- 앱 이름 변경
- 로고 URL 설정
- 테마 색상 변경

### 템플릿 관리
- Word/PPT 템플릿 업로드
- 기본/커스텀 템플릿 관리

### 감사 로그
- 사용자 활동 기록 조회
- 활동 유형/리소스 필터링

### 데이터 관리
- 전체 데이터 JSON 내보내기 (백업)
- JSON 파일에서 데이터 가져오기 (복구)
